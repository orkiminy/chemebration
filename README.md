# Chemabration

An AI-powered study platform helping Organic Chemistry students prepare for exams through question-based practice, synthesis computation, and an AI Q&A search.

Co-founded as a personal startup. I serve as the sole engineer, responsible for the full stack and UI/UX design.

##  Tech Stack
- **Frontend:** React (Create React App), React Router
- **Styling:** Custom CSS
- **Backend & Database:** Firebase (Firestore, Firebase Auth)
- **Hosting:** Vercel
- **Fonts:** Inter (Google Fonts)

##  Features
- Question-based practice with organized reaction rules and student-facing study flows
- Synthesis computation for working through reaction pathways
- AI-powered Q&A search to help students find answers fast
- Authenticated user experience with persistent progress
